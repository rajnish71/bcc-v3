Filter bar sticks below the 68px nav on scroll so the grid is always accessible with filters in view. Genre pills scroll horizontally when they exceed viewport. All filters (genre, event, sort) are combinable — API query string builds from the active state of each control. Grid/List toggle switches the layout mode.

Justified grid — each photo retains its original aspect ratio. Each row is proportionally scaled so all photos fill the full 1200px content width with 8px gaps. Row height varies by row content. Algorithm: Flickr justified-layout (or equivalent). No CSS object-fit cropping applied. 5 rows shown (22 photos). Load More triggers paginated API fetch — not infinite scroll, to keep footer accessible.
