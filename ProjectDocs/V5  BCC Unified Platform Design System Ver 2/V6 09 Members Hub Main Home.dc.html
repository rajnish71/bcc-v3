05

Specification

SECTION ANNOTATIONS
① WelcomeBand — time-aware greeting. Member tier + profile link. Sole gold CTA for the view. Hairline base rule. On day one the band carries no CTA (moved to empty-state frame).
② AttentionRow — max 3 computed items. Deadlines, renewals, upcoming registrations. Hairline top + bottom. Disappears completely when no items — no empty state placeholder.
③ RecentWork — 5 newest photographs at 4:5. Public/Pending badge. Dashed upload tile at end. "YOUR LATEST WORK" kicker with Portfolio link.
④ Happening At The Club — announcement card from committee. Event rows: date pillar · 3:2 image · badge + title + venue · registration state.
⑤ Journey Numbers — lowest visual priority. Mono figures on hairline grid. Hidden if all zero (new member). No animation hook required at MVP.
VISUAL HIERARCHY
1. Photography (55% visual area)
2. Time-aware greeting
3. Gold Upload CTA
4. Attention items (if present)
5. Happening At The Club
6. Academy progress
7. Membership Card
8. Journey Numbers
9. Left rail (wayfinding)
10. Site header (public exit)
Gold appears twice per view: active dot in rail + page CTA. Never a third instance. Shell (rail, header) stays ink-600/400 on ivory — never competes with content.

COMPONENT TREE
HubLayout (role ≥ member)
├ SiteHeader {session, member}
└ HubSidebar {hubNav[], activeRoute} — see V6 12
    [≥1200px rail · 641–1199 HubTabs · ≤640 HubMobileBar]
  └ Workspace (this document)
    ├ WelcomeBand {member, timeOfDay}
    ├ AttentionRow? {items[]}
    ├ RecentWork {photos[], uploadLink}
    ├ HappeningAtTheClub {events[], notices[]}
    ├ JourneyNumbers? {stats}
    └ PersonalStack
      ├ AcademyContinue? {progress}
      ├ MembershipCard {member, tier}
      └ JourneyStrip? {milestones[]}
SiteFooter
? = conditional
AttentionRow renders only when {items.length > 0}. JourneyNumbers renders only when member has activity. AcademyContinue renders only when {progress} exists. JourneyStrip renders only when {milestones.filter(m => m.value > 0).length > 0}.
Data contracts
WelcomeBand reads {member.firstName, member.tier, member.since, member.publicProfileUrl}. AttentionRow reads {member.computed.flags[]}. RecentWork reads {member.photos.slice(0,5)}.
Auth gate
All HubShell routes require role ≥ "member". SiteHeader Member Hub button navigates here. Unauthenticated redirect → /signin.
