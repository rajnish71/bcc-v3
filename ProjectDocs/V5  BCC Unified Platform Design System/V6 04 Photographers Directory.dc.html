MEM-006 Constitutional Constraints

Standalone annotation panel. Four constraints governing all public photographer directory surfaces. Non-negotiable.

MEM-006 · Constitutional Constraints Panel · /photographers/
1.
PRIVATE founding members must never appear in this directory
PRIVATE founding members (serial numbers 00002–00007) must never appear in this directory or any public photographer grid surface. Exclusion is enforced at the API layer via profile_visibility = PRIVATE filter. No design workaround or exception is permitted.
2.
Class filter is for directory lookup only — not a membership promotion surface
Filter tabs expose only the non-constitutional classes: All · Basic · Student · Individual · Life. Full, Patron, and Founding Member are excluded from the filter — those are constitutional classes and must not appear as selectable filter options. Tabs shown must not include any joining CTA, application link, upgrade pathway, or promotional copy attached to any tab, its label, or its results set.
3.
No joining CTA, application link, or upgrade pathway anywhere on this page
This page must not contain any element interpretable as a membership promotion, sign-up prompt, or upgrade invitation. PageHero: no CTA. Cards: no CTA. Filter tabs: no CTA. Future revisions must not add such elements without formal Design Authority review and constitutional amendment.
4.
Basic Member badge: directory and cards only — not on individual profile page
The Basic Member class badge may appear on the filter tab strip (lookup identifier) and on PhotographerCards in this directory grid. It must NOT appear on the individual Photographer Profile page (V6 05, route /photographers/[username]). Profile pages show no membership class badge for Basic members.

MEM-006 CONSTITUTIONAL CONSTRAINTS · V6 04 Photographers Directory · BCC Design Authority · July 2026
