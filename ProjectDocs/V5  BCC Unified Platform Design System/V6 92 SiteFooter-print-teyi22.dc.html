SiteFooter

Single reusable footer component for the complete BCC Unified Platform. Three-row architecture — Trust & Community, four-column navigation, copyright bar. Desktop 1280px and Mobile 390px. Every public and Member Hub page uses this footer without modification.

Inherits V6 00 BCC Design Principles and V6 91 SiteHeader. Design reference only — no implementation, no tokens, no page content.

Interaction Notes

The footer is intentionally low-interaction. No dropdowns, no toggles, no expandable states. Interaction is limited to link hover, social icon hover, and keyboard navigation.

06

Accessibility

WCAG 2.1 AA compliance required. Dark surface with appropriately lightened text ensures readable contrast. Navigation structure uses semantic landmarks and grouped lists.

Landmark Role
<footer role="contentinfo">
One per page · No nesting
Direct child of <body> or main layout
aria-label: "Site footer"
Matches screen reader landmark list
Navigation Semantics
Each column: <nav> with aria-label
"Navigate" · "Explore" · "Guidelines"
Links within: <ul><li><a>
Column heading: <h3> visually styled
Not aria-hidden — screen readers read
Colour Contrast
Nav links #C4BDB3 on #1A1816
Ratio: ≥ 4.6:1 ✓ AA
Hover #FFFFFF on #1A1816
Ratio: ≥ 13:1 ✓ AAA
Copyright #57514B on #141210
Ratio: ≥ 3.2:1 — Large text ✓
Screen Reader Considerations
Social icons: aria-label="Instagram"
(etc. per platform, not SVG title)
"SOON" badge: aria-hidden="true"
Link itself: aria-disabled="true"
FIP badge: treated as figure,
aria-label="Affiliated with FIP"
Logo in footer: alt="Bhopal Camera Club"
Copyright year: no special treatment
Focus Order & Skip Navigation
