08

MEM-006 Constitutional Constraints

Four non-negotiable constraints governing this page. No design decision may override these. Constitutional mandates enforced at both API and design layers.

MEM-006 — Photographer Profile · Four Constraints · Non-Negotiable
1 — PRIVATE EXCLUSION
PRIVATE members never appear on this route. The API enforces this at the data layer — getStaticPaths only receives PUBLIC profiles. No design bypass is possible or permitted. If a member switches to PRIVATE, the page is removed at the next build cycle.
2 — CLASS BADGE RULE
Constitutional class badges are permitted on this page. The Basic badge is never shown. Basic is the default registration state, not a membership class. Badge appears only if the member holds Associate, Fellow, or higher. Sparse profiles with no class show no badge.
3 — NO MEMBERSHIP CTA
No joining CTA, membership promotion, or upgrade pathway anywhere on this page. Photographer profiles are editorial — they showcase member work. Conversion actions exist on Home and /join only. No exception for Sparse profiles.
4 — FOUNDING MEMBER BADGE
The ★ FOUNDING MEMBER serial badge shows only on PUBLIC profiles where is_founding_member is true. Currently only serial #00001 is public. Badge: #141210 background · gold border · gold text. No other profile variant receives this treatment.
