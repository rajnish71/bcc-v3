07

Specification

SECTION ANNOTATIONS
① Page Header — title, photo count, selected count, quiet public-portfolio link. No gold CTA here — the CTA lives only in the empty state.
② Toolbar — DirectoryToolbar vocabulary: search, member's-own-genres chips, Status, Sort, Grid/List toggle.
③ Portfolio Wall — MattedThumb cards at native aspect ratio. Featured flag top-left, status chip on the label row, quiet hover controls, selection ring.
④ SelectionBar — slides in above the wall on first selection. Batch ops: Feature, Visibility, Move, Delete. Delete is the only destructive action.
⑤ Inspector Panel — right-edge contextual overlay on desktop, full-screen sheet on mobile. Never a permanent third column.
INFORMATION HIERARCHY
1. Photography (the wall)
2. Selection state (gold ring + check)
3. Toolbar (quiet, always available)
4. Page Header metadata
5. SelectionBar (appears only when needed)
6. Inspector fields (opened intentionally)
7. Left rail (wayfinding)
8. Site header (public exit)
Editing tools stay visually quiet until a mat is hovered or selected — this is the curation room, not an admin dashboard.
SELECTION BEHAVIOUR
Desktop/tablet — click a mat to toggle selection; SelectionBar appears on first selection, disappears at zero.

Mobile — long-press enters selection mode (haptic feedback + gold ring); subsequent taps toggle. Tapping empty space exits selection mode.

Delete — always requires a confirmation step; the only destructive batch action.
RESPONSIVE + CONDITIONAL
≥1200 4-col wall · inspector overlay
641–1199 3-col wall · inspector sheet
≤640 2-col wall · long-press · full-screen inspector
Conditional rendering
SelectionBar: hidden when selection count = 0
Toolbar/SelectionBar: hidden entirely on the empty state
Featured flag: renders only when photo.featured = true
Empty state
Single gold CTA — Upload Photograph — replaces the whole wall + toolbar.
COMPONENT TREE
HubLayout (role ≥ member)
├ SiteHeader {session, member}
├ HubSidebar {hubNav[], activeRoute:"portfolio"}
  ├ HubRail [≥1200px]
  ├ HubTabs [641–1199px]
  └ HubMobileBar [≤640px]
└ HubShell
  └ HubMain
    ├ PageHeader {photoCount, selectedCount}
    ├ DirectoryToolbar {search, genres[], status, sort, view}
    ├ SelectionBar? {selection[]}
    ├ PortfolioWall {photos[], view}
    │  └ MattedThumb ×n {photo, selected, featured}
    ├ InspectorPanel? {photo, prev/next}
    └ EmptyState? {ctaOnly:true}
SiteFooter
? = conditional
SelectionBar renders only when {selection.length > 0}. InspectorPanel renders only when a single photo is opened. EmptyState renders only when {photos.length === 0}, replacing Toolbar + Wall entirely.
Data contracts
PageHeader reads {member.photos.length, selection.length, member.publicProfileUrl}. PortfolioWall reads {member.photos[]} filtered by toolbar state. InspectorPanel reads {photo.caption, description, exhibitionLabel, exif, visibility, focalPoint}.
Auth gate
Inherits HubLayout per HUB-ARCH-001 — role ≥ "member" required. Unauthenticated redirect → /auth/signin.
