How Viewing Context is resolved
When the user taps a photo in a Container (Portfolio, Story, Collection, Activity, Contest), the Container passes its context via query param or navigation state: e.g. /showcase/photos/8427?ctx=portfolio&ctxId=rajnish-khare. The Canonical Photo page reads this to render the correct breadcrumbs and load prev/next IDs from the Container's ordered list. If no context param is present, context defaults to Showcase.
Canonical URL — always the same
Regardless of which Container the photo was opened from, the URL in the browser address bar is always /showcase/photos/8427. The context query param is auxiliary navigation state — it is not the canonical identity of the photograph. Sharing the URL always shares the Canonical URL. (PHOTO-ARCH-001 Principle 5)

Full-Screen Lightbox Viewer

Users may open the photograph in a dedicated full-screen Lightbox viewer — a distraction-free overlay that fills the browser viewport. This is a viewing behaviour only. It does not alter the Canonical URL, photo identity, or any architectural owner. All PHOTO-ARCH-001 principles remain invariant.

Trigger — How the Lightbox is Opened
The Lightbox is triggered by clicking or tapping the photograph in the PhotoHero zone. An ⛶ Expand affordance appears on hover (desktop) or as a persistent icon overlay (mobile). The Lightbox opens over the current page — the URL does not change.

Photography Rule — Preserved
Natural aspect ratio is maintained at all zoom levels. The image is never cropped or distorted. Dark surround (#0A0908) fills viewport area outside the image boundary — identical to the PhotoHero rule. PHOTO-ARCH-001 compliant.
URL Invariance — PHOTO-ARCH-001 P5
Opening the Lightbox does not change the browser URL. The Canonical URL /showcase/photos/{photoId} remains in the address bar. Closing returns the user to the page without navigation.
Accessibility
Focus is trapped in the Lightbox when open. Esc always closes. Close button is keyboard-reachable. role="dialog" · aria-modal="true". Body scroll locked while open.
