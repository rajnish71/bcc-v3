The Gold CTA Rule
There is exactly one gold gradient CTA per view. Its scarcity is its authority. When the user sees gold, the message is unambiguous: this is the primary action. A second gold CTA destroys this signal entirely — the system produces confusion, not clarity.

EDITORIAL FEATURE IMAGE — Container-width · aspect-ratio:16/9 · object-fit:cover · Left text panel 50% · gradient overlay · Admin-selectable (16:9 filter) · Independent of Hero Spotlight selection
16:9 Standard — Both Components
Both the Hero Spotlight and the Editorial Feature Image use the 16:9 standard. The CMS stores width and height dimensions on every uploaded photo record and derives aspect ratio from these values. Administrators see only 16:9-eligible photographs in the selector for both components. Dimensions are extracted from the final uploaded image, not the original RAW file — photographers crop before uploading; only the uploaded image determines eligibility.

Elevation & Borders

Three elevation levels maximum. No level may be stacked with another. Level 3 is reserved exclusively for hero imagery. Hairlines define structure — shadows add depth only when the element needs to float above the surface.

Hairline System

Good vs Bad Usage

Common violations and their correct implementations. These are the most frequent design decisions that either reinforce or erode the Refined Editorial Luxury identity. Ref: V6 99 §17

Correct — One Gold CTA
Gold for the primary action. Ghost for secondary. The hierarchy is unambiguous.
Wrong — Two Gold CTAs
Two gold CTAs destroy the signal. The user cannot tell which action matters.
Correct — Natural Ratio
16:9
3:4
3:2
Justified layout: shared row height, natural widths. Photography is respected.
Wrong — Forced Crop
cropped 16:9
cropped 3:4
cropped 3:2
object-fit:cover forces identical heights. All photographs are mutilated. Forbidden.
Correct — Max-Width Body

Body copy is constrained to 680px. The remaining white space is not wasted — it is the editorial silence that frames the content and creates the luxury reading experience.

max-width: 680px · Inter 400 · lh 1.75
Wrong — Full-Width Body Text

Body copy spanning the full container width becomes extremely difficult to read. The eye has to travel too far on each line return. This is a reading comfort failure that also destroys the luxury editorial feeling by filling every available pixel.

Full-width text — never do this
Correct — Square Card
Card Title
Square corners. Hairline border. e-1 shadow. This is the BCC card.
Wrong — Rounded + Accent Border
Card Title
Left accent border + heavy radius = web design trope. Not BCC.

Accessibility Reference

Accessibility is not a layer applied after design — it is embedded in the design language. The warm, high-contrast ink system, generous leading, and visible focus states are design decisions that serve all users. Ref: V6 99 §15

Contrast Ratios
#141210 on #FAF8F4: 17.6:1 ✓
#57514B on white: 7.5:1 ✓
#A8843C on white: 3.7:1 ✓ (lg)
Focus States
2px gold outline / 2px offset. Never remove focus rings without replacement.
Touch Targets
44px
Minimum touch target
All interactive elements min 44×44px. Navigation items, avatar, CTAs all meet this.
prefers-reduced-motion
All GSAP animations respect this. Transforms off, opacity transitions retained.
