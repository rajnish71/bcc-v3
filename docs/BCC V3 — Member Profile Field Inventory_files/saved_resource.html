Member Profile
Field Inventory

A complete inventory of every profile data field available to the V6 13 Member Profile page, classified by category and annotated with schema status and current data population. Intended for use during wireframe design so layout decisions are grounded in real platform capabilities, not assumptions.

  • Schema basis migrations 0001–0041
  • Phase D Batch D1 complete
  • Date 2026-07-11
  • Members in DB 19 users (17 active)
Status key
Schema + Data Column or table exists; real data is populated for relevant members now.
Schema, No Data Column or table exists; data migration is pending (Batch D2 or D3).
No V3 Schema Field exists in legacy database but no V3 column or table has been created yet. Requires a schema migration before it can be used.
Derived No dedicated column — value is computed from other tables at query time.
Deferred Explicitly deferred to a future phase; no V3 schema target in Phase D.
Discarded Permanently excluded from migration; universally empty or invalid in legacy.
